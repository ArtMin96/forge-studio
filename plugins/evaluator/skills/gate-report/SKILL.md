# Gate Report: Session Quality Summary

Review the current session for quality warnings generated by hooks and produce a summary.

## Process

1. Check for PHPStan/Larastan warnings generated during the session
2. Check for migration files without `down()` methods
3. Check for any formatting issues (run Pint `--test`)
4. Check git diff for common issues:
   - `dd()` or `dump()` left in code
   - `console.log` left in code
   - `TODO` or `FIXME` in changed files
   - Hardcoded credentials or API keys
5. **Policy-gateway activity** — grep `.claude/lineage/ledger.jsonl` for `policy-block` and `sensitive-op-audit` entries since the last commit; report counts per label.
6. **Features status** — if `.claude/features.json` exists, summarize pending/in_progress/done; flag if any `pending` items are mentioned in the current diff (suggesting incomplete work).
7. **Gate file** — if `.claude/gate/features.json` exists (written by /verify), include the pass/fail summary.
8. Run a final quick healthcheck.

## Output

```text
GATE REPORT
===========
Static Analysis:  [X warnings]
Migration Safety: [X missing down() methods]
Debug Artifacts:  [X dd()/dump()/console.log found]
TODOs Added:      [X new TODO/FIXME in diff]
Security:         [X potential credential leaks]
Formatting:       [X files need formatting]
Policy blocks:    [X secret-detected + Y injection + Z sensitive-op-audit since last commit]
Features:         [P pending / I in_progress / D done]
/verify gate:     [PASS N/N | FAIL m of N | NOT RUN]
---
READY TO COMMIT:  [Yes / No — fix these first]
```

Only report actual findings. No generic advice.

## Examples

### Example 1: clean session

Input:
```text
PHPStan run:        no errors
Pint --test:        clean
git diff:           app/Services/Order.php, tests/OrderTest.php
Policy ledger since last commit: 0 entries
features.json:      3 done / 0 in_progress / 0 pending
.claude/gate/features.json: PASS 3/3
```

Output:
```text
GATE REPORT
===========
Static Analysis:  0 warnings
Migration Safety: n/a (no migrations changed)
Debug Artifacts:  0
TODOs Added:      0
Security:         0
Formatting:       0
Policy blocks:    0
Features:         0 pending / 0 in_progress / 3 done
/verify gate:     PASS 3/3
---
READY TO COMMIT:  Yes
```

### Example 2: dirty session

Input:
```text
PHPStan run:        2 errors in app/Http/Controllers/AuthController.php
Pint --test:        1 file needs formatting
git diff: app/Http/Controllers/AuthController.php has dd($user)
Policy ledger since last commit: 1 secret-detected entry (file: .env.local)
features.json:      1 done / 1 in_progress / 1 pending; pending item touches AuthController
.claude/gate/features.json: NOT RUN
```

Output:
```text
GATE REPORT
===========
Static Analysis:  2 warnings (app/Http/Controllers/AuthController.php)
Migration Safety: n/a
Debug Artifacts:  1 (dd() at AuthController.php:54)
TODOs Added:      0
Security:         1 secret-detected (.env.local)
Formatting:       1 file
Policy blocks:    1 secret-detected
Features:         1 pending / 1 in_progress / 1 done — pending item overlaps current diff
/verify gate:     NOT RUN
---
READY TO COMMIT:  No — strip dd(), run /verify, redact .env.local before staging
```
