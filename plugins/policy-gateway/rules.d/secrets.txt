# Secrets regex patterns (POSIX extended). One pattern per line.
# Lines starting with '#' and empty lines are ignored.
# Each pattern is tagged with a label after '|' (used in ledger entries).
# Evolvable via SEPL: /evolve can propose additions here.
#
# AWS access key
AKIA[0-9A-Z]{16}|aws-access-key
# GitHub classic personal access token
ghp_[A-Za-z0-9]{36}|github-pat
# GitHub fine-grained PAT
github_pat_[A-Za-z0-9_]{80,}|github-fg-pat
# OpenAI / Anthropic-style
sk-[A-Za-z0-9]{20,}|api-key-sk
# Slack bot/user token
xox[baprs]-[A-Za-z0-9-]{10,}|slack-token
# Google API key
AIza[0-9A-Za-z_-]{35}|google-api-key
# Private key headers
-----BEGIN (RSA |EC |OPENSSH |DSA )?PRIVATE KEY-----|private-key
# JWT (3 base64url segments)
eyJ[A-Za-z0-9_-]{10,}\.[A-Za-z0-9_-]{10,}\.[A-Za-z0-9_-]{10,}|jwt
