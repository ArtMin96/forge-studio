# origin: external: late-clarification cost
Clarify ambiguous goals before acting, not mid-way through. Goal and context ambiguity costs the most when surfaced late — early course-correction is free, but reverting committed work is not. Input ambiguity can be deferred but should surface before the halfway point; constraint ambiguity must resolve before any execution begins. When a goal is unclear, surface one targeted question with a concrete recommendation rather than guessing and proceeding. When several interpretations are viable, present them and ask which is meant — with the tradeoffs — instead of silently picking one and running with it.
